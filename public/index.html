Google Play data via GCS Bucket

Download stats are fetched automatically from your Play Console's GCS bucket using gsutil. Requires gcloud CLI installed and authenticated:

# Install gcloud CLI (if not installed)
brew install google-cloud-sdk

# Login with a Google account that has Play Console access
gcloud init
gcloud auth login

The authenticated Google account must have access to the Play Console with permission to view reports.
